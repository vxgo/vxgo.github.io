Mac 实用小工具

Written by ehlxr with ♥ on March 30, 2018 in 开发工具 | 条评论

    • 收费
    • 免费

记录备忘一下 Mac 系统下一些实用的小工具

收费

  • Bartender : 菜单栏管理工具，可以将菜单放进 Bartender 的二级菜单
  • SizeUp : 实用分屏小工具
  • Proxifier : 配合 ss/ssr 可以实现真正的全局代理软件
  • iStat Menus : 能够在系统菜单栏实时监控 CPU、内存、硬盘、网络、温度、电池以及系统时间等
  • CleanMyMac : 系统清理工具，可以清除 mac 系统多余的语言包、系统缓存、应用程序等
  • KeyCue : 快捷键辅助工具，忘记一些快捷键的时候按住 Command 就会出现快捷键大全的菜单
  • Araxis Merge : 可视化文件合并以及数据同步工具

免费

  • Alfred : Mac 效率神器，Spotlight 完美替代产品（需要付费购买部分功能）
  • iTerm2 : 是一款完全免费的，专为 Mac OS 用户打造的命令行应用
  • Typora : 极致简洁的 markdown 编辑器
  • Kap : 免费开源的屏幕录像软件
  • Karabiner Elements : 修改键位映射软件
  • Gas Mask : 免费开源的 hosts 管理软件
  • KeePassX : 免费开源的密码管理软件
  • Keka : 丑到极致的压缩、解压软件（不换图标不忍直视）
  • Dr. Unarchiver : 功能强大的解压软件
