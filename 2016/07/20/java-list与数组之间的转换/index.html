Java List与数组之间的转换

Written by ehlxr with ♥ on July 20, 2016 in Java开发技术 | 条评论

      • 1. 数组转换为List
      • 2 .List转换为数组

1. 数组转换为List

String[] arr = new String[] {"str1", "str2"};
List<String> list = Arrays.asList(arr);

2 .List转换为数组

List<String> list = new ArrayList<String>();
list.add("str1");
list.add("str2");
int size = list.size();
String[] arr = list.toArray(new String[size]);
